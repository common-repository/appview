== Description ==

AppView is a plugin that allows users to easily add their website to their smartphone's home screen. With this plugin, users can customize the favicon and sitename that will appear on the home screen when their website is saved. The plugin includes an admin settings page where users can enter the URL for their favicon and set their desired sitename. Once the settings are saved, the plugin automatically adds the necessary meta tags to the head section of the website to ensure it is displayed properly on the user's smartphone home screen. This plugin is ideal for website owners who want to provide their users with an easy and convenient way to access their website from their mobile devices.



== Frequently Asked Questions ==

= How does this plugin work? =

This plugin allows users to add your website to their smartphone's home screen, giving it a native-like feel. Once added, your website will launch in full-screen mode and without the browser's navigation bar, making it appear as an app. Additionally, you can customize the appearance of the app by providing a favicon and a sitename.

= How do I install this plugin? =

You can install this plugin by following these steps:

    Log in to your WordPress dashboard.
    Go to "Plugins" → "Add New".
    Search for "AppView".
    Click on "Install Now" and then "Activate".

Alternatively, you can download the plugin from the WordPress repository and upload it manually to your website.

= Where can I find the plugin settings? =

The plugin settings can be accessed from the WordPress dashboard under "Settings" → "AppView".

= Can I customize the appearance of the app? =

Yes, you can customize the appearance of the app by providing a favicon and a sitename in the plugin settings.

= Will this plugin work on all smartphones? =

This plugin is designed to work on iOS and Android devices. However, some older devices or browsers may not support the necessary features for the app to work as intended.

= Can I uninstall the plugin without affecting my website? =

Yes, you can safely uninstall the plugin without affecting your website's functionality. However, any customizations made to the app's appearance (such as the favicon or sitename) will be removed.
